# Explainers Module Reference

The explainers package lives at [`../src/explainers/`](../src/explainers/).

## Classes

- `CurveFitExplainer`: Summarizes curve calibration quality, fit errors, and parameter intuition.
- `RiskScenarioExplainer`: Converts scenario shocks into impact narratives for PnL and risk factors.
- `PolicyNarrativeExplainer`: Frames model results in macro/policy language for stakeholder communication.
